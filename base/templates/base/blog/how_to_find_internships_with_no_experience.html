The 5-step playbook to get internships with no experience

StepWhat to doOutcome
1) Define a lane Pick one target (marketing • finance • software • design • ops). Clear keywords and companies to chase.
2) Build tiny proof Ship a 1–2 day project aligned to the lane. A link that replaces “experience.”
3) Apply widely (smart) 50–150 targeted apps; prioritize rolling & SMBs. Interviews from volume + fit.
4) Direct outreach Short, specific messages with a helpful idea. Warm intros and hidden openings.
5) Iterate weekly Improve project, resume keywords, and outreach based on responses. Compounding conversion gains.

Outreach templates (email & LinkedIn)

Keep it under 120 words. Be specific, offer help, and include one proof link.

Cold email — Marketing/Content

Subject: Student help on short-form video (example inside)

Hi {{ Name }}, I’m a {{ year }} at {{ University }}. I liked your {{ product/campaign }} and noticed {{ quick observation }}.
If helpful, I drafted 5 TikTok/IG Reel ideas with hooks + captions for {{ brand }} (doc below).
Could I intern part-time this summer to help with content & analytics? Here’s a 30-sec reel I made: {{ link }}.
Thanks for considering — happy to start with a 2-week trial.

{{ Your Name }} · {{ LinkedIn }} · {{ Portfolio }}


Cold email — Data/Operations


Subject: Free mini-dashboard for {{ Team }}

Hi {{ Name }}, I pulled {{ non-sensitive public data }} and built a small dashboard (screenshot + link).
If it’s useful, I’d love to intern and maintain weekly reports, clean CSVs, and document processes.
Available {{ months }}; tools: Excel (PivotTables), Python (pandas), Google Looker Studio.

{{ Your Name }} · {{ LinkedIn }} · {{ GitHub }}


LinkedIn DM (after a like/comment)


Hey {{ Name }} — loved your post about {{ topic }}. I’m a {{ year/major }} learning {{ tools }}.

Built a tiny {{ project }} that might help {{ their team }} (link). Open to a short internship?

Common mistakes that block offers


    
  
  • Waiting for experience: Build tiny proof now; experience follows output.
    • 
  
  • Generic outreach: Replace “I’m passionate” with a specific idea or mini-deliverable.
    • 
  
  • Low volume: Send too few apps. Aim for steady, targeted volume.
    • 
  
  • No follow-up: Follow up once after 5–7 days with a small new insight or improvement.

FAQ


What if I’m a first-year student?


Target insight programs, campus roles, nonprofits, and local businesses. Your goal is a linkable proof project and a good reference.



How long should I spend on a proof project?


1–2 days to ship v1. Improve based on feedback. Keep scope small and value obvious.



How many people should I message weekly?


10–20 targeted outreach messages + 10–20 tailored applications. Track replies in a simple sheet (company, contact, status, next step).
